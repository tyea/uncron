# Uncron

## About

Uncron is a Laravel package for running scheduled commands without Cron.

It allows you to run them in the foreground, just like queued jobs.

## Requirements

* PHP >= 7.0
* Laravel >= 5.5

## Installation

```
composer require tyea/uncron
```

## Usage

```
php artisan schedule:repeat
```
